Jiffies CSS

Usage

  • Download and save jiffies-css-bundle.min.css in your project.
  • Add overrides to custom properties in a :root {} block.
  • For appropriate semantic markup, view the source of this file.
  • Sample customization block: 
    :root {
  --color-primary-hue: deg;
  --sizing:  1;
  
  
  
}
